Asteroids

A 2D recreation of the classic game.
Made in unity using C#.

About

Asteroids is a solo project I made for one of my classes. It is a recreation of the classic game using unity. The purpose of the project was to test game object interactions, manager patterns and physics in unity.

Design Pattern

The game was made using the manager pattern with help of singletons as the main architecture. One of the main purposes of the project was to use this type of architecture to properly control the flow and interaction of the game. Rigidbody and forces were used to modify the movement of the ship. This decision is so the ship feels more like an actual moving object in space and for the player to control it like an actual spaceship.

Singletons

Using the manager pattern architecture with singletons also helps with controlling changes in the UI, by creating a singleton UI manager, one call from anywhere can notify the manager of any changes in entities, which facilitates the comunication of all classes
An example of this is the UIManager class, with the singleton architecture I can update the UI everytime the player dies and take one rocket image away. Another place I can do this is whenever a bullet hits an asteroid the score will go up.

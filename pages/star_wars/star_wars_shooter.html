Star Wars Shooter

A 3D OpenGL shooter

About

The star wars shooter is a 3D shooter made in OpenGL created using C++ using OpenGL. The purpose of the project was to make a game without using any game engine, simulating collision and drawing basic shapes using the OpenGl library.

Models

The game was made using texture wrapping and individually rendered cubes and basic shapes.

Enemy Behaviour

Three main enemies were created, design to encourage movement and a state of alert:
-Enemy 1 acts as a fast moving entity that slows down once the player is near to give them a chance to escape.
-Enemy 2 has a constant following of the player never accelerating or slowing down to make as a constant threat.
-Enemy 3 is a slow moving entity that will accelerate once they're close to the player, this will entice the player to never lower their guard even with slow enemies

Each enemy is designed to be a threat to the player to keep them stimulated and moving.

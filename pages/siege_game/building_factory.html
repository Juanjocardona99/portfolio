Building Factory

Grid drawing

The building factory is a tool I created in order to facilitate creating new buildings for the game. It uses tilemaps for drawing and a script that transforms tilemaps into png sprites that can be used for prefabs or as item icons.

By taking the array of tiles in the tilemap and locating the biggest and smallest point I can know the area the image will take up. Now with this i can create a texture loop over all the tiles and add them to the new texture, then exporting it as a png that can be used properly.

After this anything drawn can be made into a png.
